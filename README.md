## What is Vocab-you-lary
Vocab-you-lary is a CRUD app that displays programming fact cards based off a coding language. 

## Features

Vocab Cards
 - Create a new entry
 - Render all entries from database onto the page 
 - Update Entries
 - Delete Entries
 - Sort Alphabetically
 - Filter entries by programming language
